css box model
- box-sizing: content-box 
  - valor padrão
  - tamanho especificado é do conteúdo
  - quando coloca padding aumenta o tamanho do elemento
- box-sizing: border-box
  - tamanho especificado é de todo o conteúdo
  - independente do pagging ele recalcula para manter o tamanho colocado
